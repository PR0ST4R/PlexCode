Getting Started with PlexCode

PlexCode is a browser-based code editor with cloud saving, live preview, and support for many languages. Think of it as a lightweight VS Code that lives in your browser.

1. Create a file — Click New File in the sidebar or dashboard. Choose a filename and language, and you'll get a starter template.

2. Edit your code — The Monaco editor (same engine as VS Code) provides syntax highlighting, autocomplete, and more.

3. Save your work — For logged-in users, your work autosaves. Press Ctrl+S to save manually. Guest users can download their files.

4. Preview HTML — For HTML files (or any file with embedded CSS/JS), click the eye icon in the toolbar to open a live preview window.

5. Manage files — Head to My Files to see all your cloud-saved files, rename or delete them, or open them for editing.

HTML Basics

HTML (HyperText Markup Language) is the structure of a web page. Tags define elements like headings, paragraphs, links, and images.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <p>This is a paragraph.</p>
    <a href="https://example.com">Link</a>
  </body>
</html>

Common tags: <h1>–<h6> for headings, <p> for paragraphs, <div> for containers, <img> for images, <ul>/<li> for lists.

CSS Basics

CSS (Cascading Style Sheets) controls the look and layout of HTML elements.

/* Select elements by tag, class, or ID */
body {
  font-family: sans-serif;
  background: #0d0d0f;
  color: #f0f0f2;
}

.container {
  max-width: 800px;
  margin: 0 auto;
  padding: 20px;
}

#title {
  font-size: 2rem;
  color: #22c55e;
}

Key properties: color, background, font-size, margin, padding, display, flex, grid.

JavaScript Basics

JavaScript adds interactivity to web pages. It runs in the browser and can manipulate HTML and CSS.

// Variables
const name = 'PlexCode';
let count = 0;

// Functions
function greet(user) {
  return `Hello, ${user}!`;
}

// DOM manipulation
const btn = document.getElementById('myBtn');
btn.addEventListener('click', () => {
  count++;
  console.log(`Clicked ${count} times`);
});

// Fetch data
async function getData(url) {
  const res = await fetch(url);
  const data = await res.json();
  return data;
}

Frequently Asked Questions

Are my files private?
Yes. PlexCode uses Row Level Security — only you can see or access your files. Even Supabase administrators cannot read your content without explicit access.

What does username@plexcode.com mean?
It's your display name inside PlexCode. It's not a real email address — we always use your actual email for authentication. This is purely cosmetic.

Can I use PlexCode as a guest?
Yes! Guest mode lets you open, edit, and download files. However, your work won't be saved to the cloud. You'll be warned if you try to leave with unsaved changes.

Which languages are supported?
HTML, CSS, JavaScript, TypeScript, JSON, Markdown, Python, Java, C++, PHP, and Plain Text — with full syntax highlighting and autocomplete via Monaco.

How does autosave work?
For logged-in users, PlexCode saves your file 1.5 seconds after you stop typing. You can also press Ctrl+S to save immediately.

Is the preview sandboxed?
Yes. The preview iframe uses the sandbox attribute, which prevents scripts from accessing your session, cookies, or the parent page.
